# C Basics and Algorithms

Сборник задач и реализаций на языке C: работа с памятью, строками, массивами, структурами данных и базовыми алгоритмами.

## Назначение

Репозиторий объединяет компактные реализации, связанные с базовыми конструкциями языка C и типовыми алгоритмическими задачами.

## Структура

- `labs/` — практические задачи по базовым конструкциям языка;
- `algorithms/` — отдельные реализации алгоритмов;
- `README.md` — обзор репозитория.

## Содержимое

- указатели и динамическая память;
- строки и массивы;
- файловый ввод-вывод;
- пользовательские структуры данных;
- сортировки, поиск, рекурсия;
- раздельная организация `.h` и `.c` файлов.

## Архитектура

```text
Исходные файлы .c
      │
      ├─ заголовочные файлы .h
      ├─ тестовые входные данные
      ▼
Сборка и запуск
      │
      ▼
Проверка корректности
```

## Сборка и запуск

Базовый вариант для отдельных задач:

```bash
gcc main.c -o app
./app
```

## Технические акценты

- ручное управление памятью;
- работа с указателями и адресами;
- модульная структура кода;
- проверка корректности на тестовых данных.

## Ограничения

- задачи оформлены как отдельные упражнения, поэтому интерфейсы между каталогами могут отличаться;
- унификация проекта возможна через единый `Makefile` и краткие описания по каждому каталогу.

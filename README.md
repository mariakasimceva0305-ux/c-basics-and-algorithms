# C Basics and Algorithms

Сборник задач и реализаций на языке C: работа с памятью, строками, массивами, структурами данных и базовыми алгоритмами.

## Назначение
Репозиторий объединяет учебные и практические реализации, связанные с базовыми конструкциями языка C и типовыми алгоритмическими задачами.

## Содержимое
- указатели и работа с динамической памятью;
- строки и массивы;
- файловый ввод-вывод;
- пользовательские структуры данных;
- сортировки, поиск, рекурсия;
- раздельная организация `.h` и `.c` файлов.

## Архитектура репозитория
```text
Исходные файлы .c
      │
      ├──► Заголовочные файлы .h
      │
      ├──► Тестовые входные данные
      │
      ▼
 Сборка и запуск
      │
      ▼
 Проверка корректности
```

## Сборка и запуск
Примеры команд зависят от конкретной задачи. Базовый вариант:
```bash
gcc main.c -o app
./app
```

## Технические акценты
- ручное управление памятью;
- внимательная работа с адресами и указателями;
- модульная структура кода;
- проверка корректности на тестовых данных.

## Ограничения и развитие
- часть задач оформлена как самостоятельные упражнения, поэтому интерфейсы между файлами могут отличаться;
- следующий шаг — добавить единый `Makefile` и краткое описание по каталогам.
